# PromptBattle Clone

Static HTML clone of [battleverse.cn](https://www.battleverse.cn/) (PromptBattle — AI 出图对战平台).

- Dark-theme masonry/waterfall image gallery
- Responsive (5 → 4 → 3 → 2 columns)
- Infinite scroll, lightbox, hover overlays
- Built with vanilla HTML/CSS/JS

Generated via the `clone-website` skill pipeline (Playwright reconnaissance + pixel-level reconstruction).

Live site: https://YIJUEYIJUE.github.io/promptbattle-clone/
