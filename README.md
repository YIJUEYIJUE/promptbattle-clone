# PromptBattle Clone

Static HTML clone of [battleverse.cn](https://www.battleverse.cn/) — PromptBattle, an AI 出图对战平台.

Features (full clone):
- Dark / light theme toggle (persisted)
- Responsive masonry gallery (5 → 4 → 3 → 2 columns) with infinite scroll
- Lightbox viewer with author info
- 参赛投稿 (submit) form with local image upload preview + live gallery insert
- 排行榜 (rankings) with 全部 / 本周 / 本月 filters
- 关于平台 (about) page
- Login / Register / WeChat-QR modals

Built with vanilla HTML/CSS/JS via the `clone-website` skill pipeline (Playwright recon + reconstruction).

Live site: https://YIJUEYIJUE.github.io/promptbattle-clone/
